# Marketing Knowledge Core

Esta pasta guarda as regras editaveis do AdIntelligence Pro.

Altere os arquivos `.json` para ajustar estrategia, thresholds, GEOs, copies, pixel, regras de escala e V1/V2/V3 sem mexer no codigo Python.

O backend le esses arquivos em tempo de execucao por meio do `KnowledgeEngine`.
